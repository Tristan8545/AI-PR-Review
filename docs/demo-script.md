# Demo 视频讲解脚本

1. 介绍项目目标：帮助开发者快速理解 GitHub PR，并发现风险代码。
2. 展示首页：输入 PR URL，说明也可以使用内置 Demo 数据。
3. 点击 Demo 分析：展示 PR 摘要、关键变化、风险代码识别、Review 建议、测试建议。
4. 展示体验功能：loading 状态、最近分析记录、一键复制 Review Markdown、文件级折叠 patch。
5. 解释架构：GitHub API 获取变更，规则分析先扫风险，分层上下文增强，DeepSeek 生成结构化建议，质量守卫过滤和兜底。
6. 说明误报与漏报控制：只关注 diff、风险分级、置信度、规则与 AI 结合、二次后处理。
7. 说明响应速度：上下文预算限制和 PR URL 级缓存。
8. 说明未来扩展：GitHub App 自动评论、仓库索引、团队规则、AI Agent 自动 Review 和自动修复建议。
